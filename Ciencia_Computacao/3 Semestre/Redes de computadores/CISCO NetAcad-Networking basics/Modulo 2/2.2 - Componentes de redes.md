## 2.2.2 Infraestrutura de Rede

>[!info] Definição de Infraestrutura
> - A **[[Infraestrutura de Rede]]** é a plataforma que suporta a rede, fornecendo o canal estável e confiável para as comunicações.
> - O caminho da mensagem pode ser simples (cabo único) ou complexo (rede global).

### Categorias de Componentes de Rede

A infraestrutura de rede é composta por três categorias principais de componentes:

1.  **[[Dispositivos Finais]]**
2.  **[[Dispositivos Intermediários]]**
3.  **[[Meios Físicos de Rede]]**

---

## 2.2.3 Dispositivos Finais

>[!note] Interface e Identificação
> - **[[Dispositivos Finais]]** são os dispositivos de rede com os quais os usuários interagem, formando a **interface** entre os usuários e a rede.
> - São a **origem** ou o **destino** de uma mensagem transmitida pela rede (também chamados de *host*).
> - Para identificar *hosts* de forma exclusiva, são utilizados **endereços** (o *host* de origem usa o endereço do *host* de destino).

### Exemplos de Dispositivos Finais

- Computadores (estações de trabalho, laptops, [[servidores de arquivo]], [[servidores Web]]).
- Impressoras de rede.
- Telefones e equipamentos de teleconferência.
- Câmeras de segurança.
- Dispositivos móveis (smartphones, tablets, PDAs, leitores de cartão, scanners).

---

## 2.2.4 Dispositivos e Meios Físicos

>[!info] Elementos Físicos (Hardware)
> - Os **Dispositivos** e **[[Meios Físicos]]** são os **elementos físicos** ou o **hardware** da rede.
> - Eles compõem os componentes visíveis e não visíveis da plataforma.

### Componentes de Hardware Visíveis

- **Dispositivos Finais:** Laptop, PC.
- **Dispositivos Intermediários:** [[Switch]], [[Roteador]], **[[Access Point Sem Fio]] (WAP)**.
- **Meios Físicos:** Cabos (usados para conectar os dispositivos).

### Componentes Não Visíveis

- **Meios Físicos Sem Fio:** As mensagens são transmitidas pelo ar utilizando **frequência de rádio invisível** ou **ondas infravermelhas**.

### Componentes de Infraestrutura em uma Rede Residencial (Exemplo)

| Categoria | Componentes Típicos |
| :--- | :--- |
| **Dispositivos Finais** | PCs, Laptops, Smartphones, Smart TVs, Consoles |
| **Dispositivos Intermediários** | Roteador Doméstico (que geralmente integra roteador, switch e WAP) |
| **Meios Físicos** | Cabos Ethernet (fio de cobre), **Pontos de Acesso Sem Fio** (Wi-Fi) |
