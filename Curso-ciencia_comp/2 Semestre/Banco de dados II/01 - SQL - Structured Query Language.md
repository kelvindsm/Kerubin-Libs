# Classificação dos comandos SQL
- DDL: Data Definition Language - Define a estrutura do banco de dados. Cria, altera e exclui tabelas, campos e esquemas.
- DML: Data Manipulation Language - Manipula dados dentro do banco de dados. Insere, atualiza, exclui e recupera dados
- DTL: Data Transaction Language - Gerencia as transações de dados. Garante a consistência e a integridade das informações
- DCL: Data Control Language - Controla o acesso e gerencia as permissões de usuários no banco de dados
## DDL - Data Definition Language
- CREATE: Cria novos objetos, como banco de dados e tabelas;
- DROP: Exclui objetos existentes, como banco de dados e tabelas;
- ALTER: Altera a estrutura de objetos existentes, adicionando, removendo ou modificando colunas, chaves e outros elementos.
## DML - Data Manipulation Language
- INSERT: Adiciona novos registros a uma tabela existente; 
- SELECT: Recupera dados de uma ou mais tabelas.
- UPDATE: Atualiza dados existentes dentro de um registro.
- DELETE: Exclui registros específicos de uma tabela.
## DDL - Data Definition Language
### "DATABASE"
- CREATE DATABASE: Cria um novo banco de dados.
- DROP DATABASE: Exclui um banco de dados.
- USE: Seleciona o banco de dados atual para trabalhar.
### "TABLE"
- CREATE TABLE: Cria uma nova tabela com colunas, tipos de dados e restrições.
- DROP TABLE: Exclui uma tabela e todos os dados nela contidos.
- ALTER TABLE: Modifica a estrutura de uma tabela.
### "COLUMN"
- ADD COLUMN: Adiciona uma nova coluna a uma tabela;
- DROP COLUMN: Exclui uma coluna de uma tabela
