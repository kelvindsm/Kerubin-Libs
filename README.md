## Sobre o Projeto

Centrado na organização de estudos acadêmicos e desenvolvimento de Hardskills, este projeto é um meio de anotar e publicar conteúdos e materiais de estudo de forma completa e diferente.

## Como Usar

1. Abra o terminal e digite `git clone https://github.com/kelvindsm/Kerubin-Libs.git` para clonar o repositório
2. Com o Obsidian, clique em "Abrir pasta como um cofre" e procure pela pasta do repositório clonado
3. Quando o Obisidian carregar o cofre, aparecerá a seguinte mensagem: "Você confia no autor deste cofre?", selecione sim (prometo que não há nada de errado :D)

> [!TIP]
> Caso tenha dúvidas a respeito do Obsidian, por favor, acesse a [página de ajuda oficial clicando aqui](https://help.obsidian.md/)
